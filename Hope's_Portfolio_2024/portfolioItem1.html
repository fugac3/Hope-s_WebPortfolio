Add to Cart App

HTML · CSS · JS · Firebase

The simple Add to app is a project built using HTML, CSS, and JavaScript, with Firebase integration for storing added products. It provides a user-friendly interface where users can input products and dynamically add them to their cart. The project features include:

  • User Input: Users can enter product names.
  • Dynamic Listing: The entered products are dynamically listed below the input section, providing a real-time overview of the added items.
  • Firebase Integration: Firebase is utilized to store the added products, ensuring data persistence.
  • Interaction: Users can interact with the listed products by tapping on them, causing them to be removed from the cart.

This project aimed to demonstrate the fundamentals of front-end development with HTML, CSS, and JavaScript, combined with Firebase for back-end functionality. It offers a streamlined user experience for adding and managing items in an online shopping cart.
